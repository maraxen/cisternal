## Cross-cutting rules (all surfaces)

1. **JSON:** `json.dumps(obj, separators=(",", ":"))` — no `sort_keys`, no indent (serde_json compact).
2. **YAML FM:** Reuse M12.2 helpers — `description: ''` for empty; tools as `- tool` at column 0 after `tools:`.
3. **Hash:** `bundle_sha256_rust()` (already shipped M12.2).
4. **Hook specs:** Rust `bundle-hash` receives **all** `hook_specs` from bridge JSON — **no** `hooks_for_surface()` filter in rust-parity lane. Legacy lane keeps L15 filtering.
5. **Dual-track:** `Emitter(rust_parity=True)` or dedicated `emit_*_rust_parity()`; default emit unchanged.
6. **Refactor:** Extract shared bits from `claude_rust.py` → `export/_rust_emit.py` (or similar): `_compact_json`, `_agent_markdown_rust`, `_skill_markdown_rust`, `_build_claude_hooks_json`, `_hook_command_rust`.

## Surface 1: Cursor

### Rust truth (`bundle_cursor.rs`)

| Path | Notes |
|------|-------|
| `.cursor-plugin/plugin.json` | Always includes `agents`, `skills` name arrays; `hooks` nested object when hooks present; compact JSON |

| `agents/{name}.agent.md` | **Omitted** when `agents_path_verified=false` (default `emit()`) |

### Legacy Python gaps (`cursor.py`)

| Gap | Legacy | Rust parity |
|-----|--------|-------------|
| Agent files | Emitted when `agent.body` non-empty | **Must omit** (match conformance digest) |
| plugin.json agents/skills keys | L17: omitted when nothing to emit | Always present when bundle has agents/skills |
| JSON | `sort_keys=True, indent=2` | Compact, insertion order: name, version, description, agents, skills, hooks?, mcpServers? |
| Hooks command | Raw `spec.script` | Raw script (no `env PRAXIA_*` wrapper in bundle adapter) |
| Event dialect | `beforeShellExecution` etc. | Same mapping — already correct in legacy |

### Legacy tests to preserve
